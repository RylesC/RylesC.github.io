CORDIC

Project Overview

As part of a course on VLSI, I implemented a 16 bit circular CORDIC algorithm on a Basys3 Artix-7 FPGA development board. This algorithm is capable of rotating a vector as well as calculating the magnitude and tan of the angle. Overall, the algorithm was successful, completing in just 32 clock cycles. The software development was done in VHDL using Vivado studio.

Features

  • Look up table for special angle values
  • ALU for bit shifting and add/subtract functions
  • Parallel RAM for x,y, and z variable storage
  • Finite state machine for user and iteration control
  • Button debouncer
  • 7-segment display driver
