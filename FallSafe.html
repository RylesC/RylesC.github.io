FallSafe

Project Overview

For my capstone project at UVic my team and I developed FallSafe. FallSafe is a radar based fall detection device which utilizes the ADALM-PLUTO, a low cost software defined radio (SDR), to detect fall events. FallSafe is designed to accurately detect falls in a single occupancy dwelling. This system also operates without the use of a camera allowing for privacy to be maintained. Frequency-modulated continuous wave (FMCW) radar was implemented in MATLAB. With this radar implementation it is possible to receive accurate Doppler-time data. This data is further enhanced through the use of a short-time Fourier transform (STFT) to increase the time resolution of the data. In order to classify fall and non-fall events, a machine learning approach was taken. Test data was collected for fall, standing, sitting, and walking events with a total of 52 falls, and 46 non-falls. A Support vector machine and deep learning architectures were applied to this test data. The SVM was implemented using manual feature extraction from the Doppler-time data. Deep learning was achieved through the use of stacked autoencoders with softmax regression. Overall, The SVM architecture achieved a 90.9% classification accuracy and the Deep learning approach achieved a classification accuracy of 93.9%. Overall, radar has the potential to be a viable solution to address the issue of fall detection.

Features

  • ADALM-PLUTO SDR
  • FMCW radar
  • Short-time Fourier transform
  • Deep learning architecture
  • 93.9% classification rate
